# 🧠 Meeting Intelligence System using Transformers

A Transformer-based NLP system that automatically analyzes meeting transcripts and generates structured meeting reports.

The system extracts:

- Discussion Topics
- Action Items
- Decisions
- Meeting Summary Insights

This project demonstrates how Large Language Model techniques can be applied to business meeting intelligence.

---

# 🚀 Features

✅ Meeting Transcript Analysis

✅ Zero-Shot Topic Classification

✅ Action Item Extraction

✅ Decision Detection

✅ Automated Meeting Reports

✅ Transformer-Based NLP

✅ Business Intelligence Automation
